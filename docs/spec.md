# ShareCheck 仕様書（MVP）

ShareCheck は、iPhone標準の写真アプリの操作感をできるだけ維持しながら、家族との iCloud 共有アルバムへの共有漏れを防ぐための補助アプリである。

本アプリは写真管理アプリではない。写真の閲覧・編集・検索・アルバム管理は写真アプリに任せ、本アプリは共有状態だけを管理する。

## 管理状態

- 未処理: ダッシュボードへ表示される。
- 共有済み: 共有したとユーザーが確定した状態。ダッシュボードには表示されない。
- 確認済み: 共有しないとユーザーが判断した状態。ダッシュボードには表示されない。

## 初回起動

初回起動時に現在日時を管理開始日時として保存する。管理開始日時より前のメディアは管理対象外とする。

## 想定利用フロー

1. ダッシュボードを見る
2. 未処理一覧を開く
3. 共有したい写真を選択する
4. iOS標準共有シートで共有する
5. 選択分を共有済みにする
6. 残りを確認済みにする
7. 未処理0件にする
